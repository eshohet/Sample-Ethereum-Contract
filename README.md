# Sample Ethereum Code

## Dependencies

1) MongoDB - `brew install mongodb`
2) Metamask, Mist, or Parity

## Installation

`npm install && npm run ethinstall`

## Running

Run the following, in order, each as a new process:

`npm run mongo`

`npm run testrpc`

`npm run deploy`

`npm run start`

## Testing

`npm run test`
